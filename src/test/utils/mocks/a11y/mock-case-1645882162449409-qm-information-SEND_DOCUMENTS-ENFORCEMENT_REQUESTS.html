Send documents Send enforcement requests

Enforcement requests cannot be uploaded using the Money claims system.

There is guidance on what to do if a defendant does not pay after a judgment (opens in a new window)

There are various options for you to enforce a judgment, you will have to fill in forms and submit them to the court.

The options for enforcing a judgment are:

  • warrant of control (opens in a new window)
  • attachment of earnings order (opens in a new window)
  • third-party debt order (opens in a new window)
  • charging order (opens in a new window)

Each option has its own fee, check the civil fees list (opens in a new window) to find the costs for the option you select.

If you need help with anything else

If you still need help with your claim, you can send a message to the court.

We try to answer messages within 10 working days, but it could take longer.
